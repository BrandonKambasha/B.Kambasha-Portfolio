Website Design

Commencing with the website creation, the design chosen for desktop display was that of having a menu option present on all the pages at the same position, with the menu disappearing into a menu button when entered into mobile version. The design chosen to fulfill this functinallity was that of having a navigation panel on the left of the webpage that contains all the menu options. The webpage will be divided into 4 different sections. The left section will be the navigation panel, a main section where the content will be displayed,a header section for the Heading which will be present on each page of the webpage and a footer section to contain a link to be able to navigate to the top of the page when clicked once reached the bottom of the page.

In the mobile version of the website, the menu options will no longer be present on a navigation panel but rather inside a hamburger icon menu present at the top left of the webpage. The header, main and footer will remain in their respective positions adjusting to fit with the width of the mobile device(800px maximum).

Across all the webpages in the desktop display the sections will be fixed in a grid layout with the naigation bar, header and footer being present in their original states on all the pages. The main section will be the only section changeing to display content accordingly to each page. In the mobile version the hamburger icon menu, the header and the footer will be in the same positions on all the pages with once again the main section the only section changeing to satisfy the selected page.

The fonts chosen for the webpages were: Roboto, Oxygen, Ubuntu, Cantarell and Calibri. These were the fonts used throughout the development of the website on all the webpages. As for the theme and the colours of the website, an overall theme colour combination of blue, white, yellow and black was used on all the webpages. A lightblue background on the header,naigation and footer sections, with white text. Yellow for the border for these sections and underlining for the Subtopics for each page.
